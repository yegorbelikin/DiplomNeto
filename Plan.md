# 1. Введение и цель проекта
Цель тестирования — проверить работоспособность основного функционала приложения, выявить критические дефекты перед запуском и убедиться, что пользователь может успешно работать с новостями, цитатами и информацией о приложении.
# 2. Границы приложения
Авторизация
- Страница Main - просмотр новостей, сортировка, фильтрация
- Страница News - редактирование, создание, удаление новостей, их сортировка и фильтрация
- Страница About - информация о приложении
- Страница Love is all - просмотр цитат о хосписе
